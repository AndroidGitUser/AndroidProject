Projektname: TimeSnapp

Beschreibung: App zum Erstellen von Zeitraffern „timelapse“ aus Fotos.
Dazu gehört das erstellen von Projekten in denen Bilder gespeichert werden.
Das Aufnehmen und abspeichern von Fotos. Die Anzeige der Fotos mit weichen Übergängen um den Zeitraffereffekt zu simulieren.

HINWEIS: Im ProjectDetailViewController kann die Variable „useTestImages“ geändert werden, um Test Bilder zu benutzen.

Kategorien: Media and Camera und Data Storage
